# About 
Built a private local AI chat system on Powerful Mac Studio. It runs three models in one interface.

Two models use MLX(OSS models for mac), and one runs through DS4 (deepseek oss model for mac) over a local HTTP API. connected everything make it compatible with a FastAPI backend and built a single chat UI to switch between models easily. No cloud inference. No external API dependency. Client data stays on the machine.

This is a strong step toward offering secure private AI solutions for clients who care deeply about data privacy.
